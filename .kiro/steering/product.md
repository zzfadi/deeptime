# DeepTime Product Overview

DeepTime is an educational Progressive Web App (PWA) that lets users explore Earth's geological history at their current location. Users can travel through 4.5 billion years of geological time, viewing prehistoric landscapes in augmented reality.

## Core Features

- **Location-Based Geology**: Real geological data from USGS Macrostrat API for any location worldwide
- **Time Travel Slider**: Scrub through millions of years of Earth's history with haptic feedback
- **AI-Powered Narratives**: Contextual descriptions generated by Google Gemini
- **AR Visualization**: View prehistoric creatures and environments via WebXR (with iOS fallback using model-viewer)
- **Offline Support**: IndexedDB caching for geological data and narratives
- **PWA**: Installable on any device with service worker support

## User Flow

1. User grants location permission or searches for a location
2. App fetches geological data and displays era cards
3. User selects an era to view details and AI-generated narrative
4. User can enter AR mode to see prehistoric creatures in their environment
5. Time slider allows navigation between geological eras with smooth transitions

## Target Users

- Students learning Earth science
- Teachers seeking educational tools
- Nature enthusiasts exploring local geology
- Anyone curious about prehistoric history
