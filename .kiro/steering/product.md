# DeepTime Product Overview

DeepTime is an educational PWA that lets users explore Earth's 4.5 billion year geological history at their current location through augmented reality.

## Core Features

- **Location-Based Geology**: Real geological data from USGS Macrostrat API for any location worldwide
- **Time Travel Slider**: Navigate through millions of years with haptic feedback at era boundaries
- **AI-Powered Content**: Contextual narratives, images, and videos generated by Google Gemini and Veo
- **AR Visualization**: View prehistoric creatures via WebXR (with iOS Quick Look fallback)
- **Offline Support**: IndexedDB caching with Firestore cross-device sync
- **PWA**: Installable on any device with service worker support

## User Flow

1. User grants location permission or searches for a location
2. App fetches geological data and displays era cards
3. User selects an era to view details and AI-generated content (text → image → video)
4. User can enter AR mode to see prehistoric creatures in their environment
5. Time slider allows navigation between geological eras with smooth transitions

## Target Users

- Students learning Earth science
- Teachers seeking educational tools
- Nature enthusiasts exploring local geology

## Key Requirements

- Content must be location-specific (include place names, local geological features)
- AI content is cached for 30 days to minimize API costs
- Graceful degradation when offline or API unavailable
- Progressive loading: text → image → video
