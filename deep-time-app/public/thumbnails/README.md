# Creature Thumbnails

This directory contains preview images for creatures shown in the card-based fallback view.

## Required Thumbnails

Each creature needs a thumbnail image:
- Format: PNG with transparency
- Size: 256x256 pixels recommended
- Naming: `{creature_id}.png` (e.g., `trex.png`)

## Required Files

- trex.png
- triceratops.png
- velociraptor.png
- brachiosaurus.png
- stegosaurus.png
- allosaurus.png
- mammoth.png
- sabertooth.png
- megatherium.png
- coelophysis.png
- dimetrodon.png
- meganeura.png

## Generating Thumbnails

Thumbnails can be generated from the 3D models using:
1. Screenshot from Blender/Three.js viewer
2. AI image generation (DALL-E, Midjourney)
3. Stock images with appropriate licensing
